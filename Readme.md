Smart Campus Information System

Introduction

The Smart Campus Information System is a Python-based mini project developed as part of the Python Programming Laboratory. The project aims to integrate multiple academic management functionalities into a single application. It demonstrates the practical implementation of Python programming concepts such as conditional statements, loops, data structures, functions, file handling, exception handling, sorting and searching algorithms, and data analytics.

Educational institutions often manage student records, course enrollments, fee information, and academic performance using multiple independent processes. This project provides a unified system that simplifies these operations through a menu-driven application. The system enables users to perform various academic management tasks efficiently while maintaining student records in a structured manner.



Objective

The primary objective of this project is to develop an integrated Smart Campus Information System by combining all modules implemented throughout the Python Programming Laboratory. The project demonstrates how individual programming concepts can be integrated into a complete application capable of handling real-world academic management tasks.

Working of the System

The application follows a menu-driven approach where the user can select different modules from the main dashboard. Each module performs a specific task related to academic management. Student information is stored and processed throughout the application, allowing different modules to interact with the same dataset.

The system supports student registration, course management, fee calculation, file operations, and data analysis. Academic records can be saved and retrieved using CSV files, while performance analytics provide meaningful insights through statistical calculations and graphical representations.

Exception handling mechanisms have been incorporated to ensure that invalid inputs, missing files, and incorrect directory paths are handled gracefully without interrupting program execution.



How to Execute the Program

Step 1: Download the Project

Download or clone the project repository to your local system.

Step 2: Install Required Libraries

Open Command Prompt or Terminal and install the required Python libraries:

pip install numpy pandas matplotlib

Step 3: Open Project Directory

Navigate to the project folder containing the Python source code.

Step 4: Run the Application

Execute the Python program using:

python main.py


Step 5: Use the Dashboard

After execution, the Smart Campus Information System dashboard will appear. Select the desired option from the menu and follow the instructions displayed on the screen.
